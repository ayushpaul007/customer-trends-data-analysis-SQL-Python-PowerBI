# 👨🏻‍💻 Customer Behavior Data Analyst Portfolio Project

This project represents a complete, industry-standard, end-to-end data analytics workflow designed to simulate real-world responsibilities of a data analyst. It covers data preparation, SQL analysis, visualization, and reporting.

---

## 📌 Project Overview

The goal of this project is to transform raw customer data into actionable business insights through:

* ✅ **Data Preparation & EDA (Python):** Clean and transform raw data
* ✅ **Data Analysis (SQL):** Run queries to extract insights
* ✅ **Visualization (Power BI):** Build interactive dashboards
* ✅ **Reporting:** Present insights and recommendations

---

## 🛠️ Tech Stack

* **Python** (Pandas, SQLAlchemy)
* **SQL Database**
* **Power BI** (Visualization & Dashboarding)

---

## 📊 Key Analysis Performed

* Top-performing products based on customer ratings
* Discount impact on purchasing behavior
* Customer purchase trends
* Aggregated metrics using SQL

---

## 🚀 Outcome

By completing this project, you will:

* Build a strong portfolio project
* Learn Python + SQL + Power BI integration
* Gain real-world analytics experience

---

## 👨‍💻 About

This project is ideal for:
